CSS Animations

A CSS animation is the process of chaning an element’s property/properties over a given duration of time without the need for JavaScript. You can change(animate) an element’s size, colour, location on screen and even rotate it, to name a few. To create a CSS animation you need to set the properties to be animated and keyframes which define how the animated element will render/display at a given time.
