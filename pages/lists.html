HTML Lists

An HTML list is a collection of related items e.g., days of the week or ingredients in a recipe. This page will illustrate the various types of lists we can create in a webpage.

Ordered Lists This is a collection of items in a particular order or steps that need to be followed or carried out in a given order. Some examples are given below. Days in a Week: Sunday Monday Tuesday Wednesday Thursday Friday Saturday Unordered Lists This is a collection of items in no specific order or preference, though the items are related. An example is a list of ingredients to make KFC Chicken. Nested Lists This is a list that contains other list(s) within it. It can contain either an ordered or unordered lists or both.
